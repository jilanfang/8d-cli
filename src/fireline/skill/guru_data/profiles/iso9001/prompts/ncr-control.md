# ncr-control — 不符合输出控制（ISO 9001:2015 第 8.7 条）

## 行业背景

ISO 9001:2015 第 8.7 条是对 8D 围堵阶段（D3）有直接指导的体系条款。它要求组织"确保不符合要求的输出得到识别和控制，以防止其非预期的使用或交付"。审核员会专门查"不符合输出的性质以及随后所采取的措施的记录，包括让步的信息"。

在真实的 8D 调查中，围堵阶段最常犯的两个错误：
1. 把不同的处置路径（返工/返修/让步/报废/降级）混为一谈——"反正都挑出来了"——然后发现让步的货没有客户书面批准，返修的产品没有重新验证。
2. 处置决定没有记录——审核员查的时候只有一张不合格品处理单，没有说明处置理由、没有客户批准证据、没有重新验证记录。

本 prompt 确保围堵下来的产品每一件的去向都有据可查。

## S1 | 发散探索 — 多角度触发器

Agent 从以下问题中选 3-4 个向用户提问。

1. **处置路径选择**："五个风险窗口（客户现场/在途/库存/WIP/上游）的不良品，每批的处置路径分别是什么？返工？返修？让步接收？报废？降级销售？不要一句话概括——每个窗口逐一说明。"

2. **返工 vs 返修区分**："你说的是返工还是返修？返工能让产品完全回到原规格——功能和尺寸都符合原图纸。返修只能恢复功能但可能不符合原规格——这种需要客户书面让步。明确说清楚。"

3. **让步接收的授权条件**："让步接收——谁批的？授权级别对不对？让步有没有明确的适用范围（仅限于这批？仅限于这个程度的不良？）让步有没有期限？让步产品有没有特殊标识以便追溯？"

4. **报废品的实物控制**："报废品是怎么处理实物的？是物理破坏了还是丢在报废区？有没有被重新混入良品的风险？报废区是隔离的还是开放的？"

5. **重新验证的完整性**："返工/返修后的产品——重新检验了吗？检验方法是不是和正常产品一样？还是简化了？如果是简化检验，依据什么判定合格？返工后的检验记录能不能和原生产批次关联？"

## S2 | 第一性原理收敛

追问链 —— 逐路径审查处置合规性。

**Layer 1 — 处置路径的法律和合同合规**
- "让步接收的产品——客户合同里有没有要求客户事先书面批准？你拿到了吗？批准的邮件/文件在哪？"
- "降级销售（比如把 A 级品当 B 级品卖）——降级后的产品是否符合 B 级的全部规格要求？还是'B 级也不要紧'的靠嘴说？降级销售有没有质量条款的风险？"

**Layer 2 — 处置后的追溯能力**
- "三个月后客户问你这批让步产品用在哪了——你能追踪到每一件的去向吗？让步产品有没有单独的批次标识？"
- "返工的产品——返工后的检验报告上能不能追溯返工之前的原批次号？如果返工后出了新问题，你能不能回溯到返工前的问题？"

**Layer 3 — 体系的处置能力**
- "你的 QMS 程序文件里有没有明确定义五种处置路径的条件和审批权限？还是每次 8D 临时决定？审核员会查：程序文件定义了没？执行按程序做了没？"
- "如果有供应链涉及——供应商端的不良品处置是不是也要走同样的 8.7 要求？供应商的让步/返工/报废——你有追踪吗？"

## S3 | 反事实挑战

1. "如果有 100 件让步接收的产品，因为没有特殊标识，混在正常产品里出货了——你现在怎么把它们找出来？如果找不出来，证明你的标识系统有漏洞。"
2. "假设返工后的产品又出了新问题（和这次 8D 不同的问题）——你的检验记录能区分'这是返工品'还是'这是正常品'吗？如果不能，你会把新问题归因到原生产过程，而实际是返工过程引入的。"
3. "如果客户一年后因这批让步产品出了安全事故要追溯——你能在多少时间内提供：让步批准记录+让步产品去向+让步产品的出厂检验报告？"

## S4 | 答案评分

评分维度：**处置路径明确性**、**授权合规性**、**追溯能力**、**重验证完整性**

| 等级 | 标准 |
|------|------|
| Insufficient | 处置路径未逐一说明；返工/返修不分；让步无书面批准；返工后无重新检验记录；报废品去向不明 |
| Basic | 五个窗口的不良品各有处置路径；返工/返修区分正确；让步有授权记录；返工后有重检验记录；报废有去向记录 |
| Excellent | 每个处置决定有依据（为什么选这个处置方式而非其他）；让步有客户书面批准+适用范围+期限；返工/返修品重检验记录与原生产批次建立了双向追溯链接；程序文件定义了五种处置路径的条件和权限；供应商端不良品处置有对应管控 |

## 纠偏原则 / 禁止项

| 禁止 | 替代方向 |
|------|---------|
| 把所有不良品统一说"返工处理" | 逐项区分：真正符合返工定义的才叫返工，否则是返修、让步或报废 |
| 让步接收不经客户书面批准 | 任何让步接收如果客户合同有要求——必须客户书面批准；没有要求也应内部有授权 |
| 返工品用更简单的方法检验 | 返工品的重检验必须至少覆盖被返工的缺陷项 + 返工过程可能影响到的其他项 |
| 报废品不做实物控制 | 物理破坏、隔离、双人见证——防止重新混入 |
| 处置记录不关联到 D3 围堵方案 | 围堵方案和处置方案是同一件事的两个阶段——围堵找到的货必须对接到处置记录 |
