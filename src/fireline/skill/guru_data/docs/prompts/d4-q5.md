# d4-q5 -- 根因验证

## 行业背景

ASQ 原则:你在 D4 找到的是"最可能的根因"，不是"已证明的根因"。两者差距是验证。MAHLE 要求主动试验验证(Active Verification):人为操控怀疑原因并观测结果变化。BSH D4 常见驳回理由:"Root cause based on assumption, not verification"。未经验证的根因进入 D5，每一分钱都可能打在错误目标上。

## S1 | 发散探索 -- 多角度触发器

1. 你声称的根因可以被观测或测量吗?如果不能，它可能只是一个无法证伪的观点，不是真根因。
2. 有哪些独立信息源支持这个根因?数据、观察、试验、历史记录 -- 来源之间是否互不依赖?
3. 能做"开/关"试验吗?消除怀疑的原因看问题是否消失，加回去看是否复现。
4. 如果涉及物理机制:能在实验室复现吗?需要什么条件?复现概率是多少?
5. 有没有其他批次满足同样的"根因条件"但没有出问题?如果有，怎么解释?是否遗漏了必要条件?

## S2 | 第一性原理收敛

**试验设计追问**:
- "'开/关'测试怎么设计?控制哪些变量?样本量够不够统计显著?"
- "如果不能做主动试验，有没有被动证据替代:历史数据趋势、事件先后顺序、自然实验?"
- "你的验证方法能不能区分'这个原因是必要条件'和'这个原因是充分条件'?"

**证据可信度追问**:
- "每条证据来源是什么?同一来源的多条数据算一个证据还是多个?"
- "有没有一条证据，一旦被推翻，整个结论就不成立?那条证据多可靠?"
- "证据链相邻环节之间有没有逻辑跳跃?"

**不确定性追问**:
- "有没有我们还不知道的，一旦知道了可能推翻当前结论?"
- "低置信度是因为证据不足、证据矛盾还是逻辑缺口?"

## S3 | 反事实挑战

1. 假设外部专家审计你的验证，他最可能攻击哪个证据?最弱的点在哪?
2. 如果给你一周和一笔预算，怎么把置信度从当前水平提到 0.9 以上?

## S4 | 答案评分

| 维度 | Insufficient | Basic | Excellent |
|------|-------------|-------|-----------|
| 验证方式 | 只有推理，无试验/数据 | 有历史数据或观察支持 | "开/关"试验结果，或历史统计相关性+因果方向确认 |
| 证据来源 | 单一来源 | 2 个以上独立来源 | 3 个以上独立来源，交叉一致无矛盾 |
| 不确定性 | 未提及 | 列出不确定性 | 列出不确定性+每个的消除方案+剩余风险量化 |
| 证伪性 | 结论无法证伪 | 可证伪但未尝试 | 已做反事实检验，经受住证伪 |

D4 铁律:验证方式无法区分"真根因"和"恰好伴随出现但无关"的，一律 Insufficient。相关性不等于因果性。

## 纠偏原则 / 禁止项

| 禁止 | 替代方向 |
|------|---------|
| 用推理代替证据 | 每句推理后跟"支持的证据是{...}"或"没有证据，需要{...}" |
| 单证据断言"根因确认" | 至少两个独立来源，不能是同一个人说的不同版本 |
| 证据描述模糊 | 必须说:什么数据、哪个时间、偏多少、和什么基线比 |
| 不报告剩余不确定性 | 问:"有什么你还不知道的?会影响 D5 吗?" |
| "试验做了，确认了"无细节 | 必须:样本量、条件、结果数据、可重复性、和现场的比对 |
| 忽略不一致证据 | 只要有一个数据点和根因矛盾就必须解释，解释不了就降级 |
