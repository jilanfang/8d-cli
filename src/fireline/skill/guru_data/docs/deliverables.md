# 岗位交付物草案指南 v0.2.1

每完成一个阶段的 S5（综合交付），生成该阶段的 markdown 草案。每个阶段有两套模板：**Basic**（满足行业最低要求）和 **Excellent**（达到标杆水平）。

## 通用原则

- 草案可直接复制到 Word/8D 系统/客户表单中。
- 不要包含内部推理过程，只保留结论。
- 对未填写的项标记"（待补充）"，不要留白。
- Excellent 版本比 Basic 版本多出的内容标注「🌟」。
